fragmentmanager
 getChildFragmentManager （母是fragment）和  getSupportFragmentManager（母是activity）
 且这个manager是一个fragment 或 一个activity 独立
 如要关闭一个fragment中嵌套的dialogfragment 可以在dialogfragment中这么写 
       Fragment prev = getParentFragment().getChildFragmentManager().findFragmentByTag("voice_recording_transfer_textdialog");
            if (prev != null) {
                DialogFragment df = (DialogFragment) prev;
                df.dismissAllowingStateLoss();
            }
注意：getParentFragment().getChildFragmentManager().findFragmentByTag 

当想从manager中删除
 getParentFragment().getChildFragmentManager().beginTransaction().remove(this);
